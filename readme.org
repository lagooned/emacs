* install

- =ln -s ./emacs ~/.emacs.d=
- =emacs &=
* usage
- evil-leader-key is SPC (much like spacemacs)
  - create env specific .custom.el by using SPC c . (default)
  - quit/restart with SPC q q and SPC q r (default)
- use which-key to explore functionality
- have fun :)
